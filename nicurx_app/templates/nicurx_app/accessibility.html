Accessibility Practices


      NICURx takes accessibility seriously when it comes to designing our web app. NICURx uses a wide array of design features to ensure that our technology is inclusive. Designed to be used by all users with any range of abilities, NICURx emphasizes an inclusive design philosophy that ensures accessibility, usability, and convenience are at the core of its functionality. We encourage anyone who finds this app inaccessible to submit email us using the 'Contact Us' link below. The following practices are implemented within this web app.


      Accessibility Design Practices

      • Writen in plain English/ simple sentences
      • Descriptive buttons/ links /headings
      • Simple colors/ Color contrast >4.5
      • Uses color, shapes, and text to convey information
      • All info published in webpage
      • Buttons in page context
      • Responsive containers useful for mobile/touchscreen devices
      • Linear and logical layout
